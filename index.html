irodoriについて

私たちの想い

子育ては一人で抱え込みがちです。周りに頼れる人がいない、相談先がわからないと感じるママの孤独や不安に寄り添いたいという思いから「irodori」は誕生しました。

3つの特徴

1

全員医療者

専門知識を持ったスタッフがサポートするため、医療的な面でも安心感があります。

2

居場所づくり

ただ話を聞くだけでなく、ママ自身がリラックスし、笑顔になれる居場所を提供します。

3

これから育つ場所

完成された場所ではなく、地域の方や参加するママたちと一緒に育てていくコミュニティです。

活動内容

irodoriは「ママがほっとできる居場所」を作るため、様々な取り組みを行っています。

子育てや心と体のケアについて、医療者の視点を取り入れた講座を行っています。

メンバー紹介

irodoriスタッフは全員が資格を持った医療関係者です。
身体のこと、心のこと、どうぞ安心してお話しください。

塚本 梓

irodori代表 / 看護師 / タイムコーディネーター

整形外科や産婦人科、看護大学での勤務経験。現在は双子を含む４児のママです。

私の経験を活かし、子育て・仕事・家庭を両立しながら、自分の人生も大切にしたい女性を応援しています。

樫本 歩

irodori副代表 / 看護師 / フェムケアタッチケア講師

急性期病院で看護師として勤務しながら、3人の子どもを育ててきました。子育てを通して地域とのつながりや居場所の大切さを実感し、現在は子ども食堂や親子向けイベントなどの子育て支援活動に取り組んでいます。

また、看護師としての経験を活かし、フェムケアやタッチケアの普及活動も行っています。

網本 頌子

産婦人科医 / ヒプノバーシング講師

産婦人科医として総合病院で働きながら、年の差兄妹３人子育て中です！

「しあわせなお産は日本をかえる！」をモットーに、妊娠中のママから子育てママと子どもの居場所づくりに向けて活動中です。

前田 沙也加

看護師 / フォトグラファー / ベビーマッサージ講師

大学病院でNICU、小児科で働いたのち小児の訪問看護師として働いてきました。

子育てする人を支えたいという思いから、ベビーマッサージ講師やフォトグラファーとしても活動しています。

中島 佳奈

助産師 / 看護師 / 元営業ウーマン

一般企業で会社員をしていましたが、長男の出産をきっかけに学校に入り直し、助産師になりました。大学病院のNICU→現在は女性病棟で産科・婦人科・周手術期の看護をしています。

たまに助産院へもお手伝いに行き、お産や産後ケア助っ人しています。男の子３人のママです。

今富 礼華

看護師 / 保健師 / 性教育アドバイザー

外科やICUなどで看護師として働いていましたが、三男出産を機に退職しました。

男の子ママになったことをきっかけに性教育を勉強し始め、現在は性教育アドバイザーとして「０歳からの性教育」などママ向け講座を主宰しています。

メンバー募集

一緒に活動しませんか？

irodoriでは、私たちの想いに共感し、一緒にママと子どものための居場所づくりを進めてくださる仲間を募集しています。

こんな人を募集しています

  • irodoriの想いに共感している方
  • 人の話を否定せずに温かく聴ける方
  • 医療者・保育士・助産師などの資格をお持ちの方歓迎
  • 資格がなくても、ママたちの支えになりたい熱意のある方
